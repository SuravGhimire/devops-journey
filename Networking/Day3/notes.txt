Networking Day 3 : DNS (Domain Name System)

What is DNS?
DNS translates domain names into IP addresses.

Example:
github.com -> IP address 

Why DNS?
Humans remember names.
Computers use IP addresses.

Website Loading Process:
------------------------
Browser
↓
DNS Lookup
↓
IP Address
↓
Connect to Server
↓
Website Opens

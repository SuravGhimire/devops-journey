( Git Stash:
Stash lets you temporarily save unfinished work.

Think of it like:
“Pause this work, I’ll come back later.” )


git stash
-Temporarily saves changes

git stash list
-Shows saved stashes

git stash apply
-Restores stash

git stash drop
-Deletes stash

In git stash , I did added a temperory note in the day2 note.txt then ,using git status , I see some modified files , then did git stash ,it Saves user's uncommitted changes,Cleans working directory , then git status was clean 
And to view stash I use git stash list , it shows some stash@{0} and to restor I use git stash apply, then git status the changes are back and lastly to remove stash , used git stash drop. 
But If i had done git add , commit and push after , git stash apply , those temporary file changes would have been permanent.

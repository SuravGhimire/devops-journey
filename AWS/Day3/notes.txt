AWS Day 3

Topic:
EC2 (Elastic Compute Cloud)

EC2 is a virtual machine running in AWS.

Uses:
- Host websites
- Run applications
- Deploy APIs
- Run databases

Benefits:
- Easy to create
- Scalable
- Pay only for usage

Think:
EC2 = Cloud Server

----------------------------------------------------------

EC2 Components

AMI:
Operating system template.

Examples:
- Ubuntu
- Amazon Linux

Instance Type:
Defines CPU and RAM.

Example:
t2.micro

Key Pair:
Used for secure SSH login.

-------------------------------------------------------------

Networking Concepts

Public IP:
Reachable from internet.

Private IP:
Used inside AWS network.

Security Group:
AWS firewall controlling traffic.

Common Ports:
22  SSH
80  HTTP
443 HTTPS

EC2 Launch Checklist

Before launching an EC2 instance:

1. Select an AMI
   Example: Ubuntu 24.04 LTS

2. Choose Instance Type
   Example: t2.micro (Free Tier)

3. Create or Select Key Pair
   Used for SSH login

4. Configure Security Group
   Allow:
   - SSH (Port 22)
   - HTTP (Port 80) if hosting websites

5. Launch Instance

After Launch:
- Get Public IP
- Connect using SSH
- Manage server remotely

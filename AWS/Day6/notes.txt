AWS Day 6

Topic:
Local AWS using Floci

Problem:
No AWS account available.

Solution:
Use Floci to emulate AWS services locally.

Benefits:
- No AWS account required
- No credit card required
- Safe local environment
- Good for AWS CLI practice

Goal:
Use AWS services locally before moving to the real AWS Cloud.


----------------------------------------------------------------------------

Why Floci?

Normally:

AWS CLI
      ↓
AWS Cloud

requires:
- AWS Account
- Internet
- Billing Account

Since I don't have an AWS account,
I will use Floci.

Floci emulates AWS services locally,
allowing me to practice AWS without using the real cloud.

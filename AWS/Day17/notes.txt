AWS Day 17

Topic:
Amazon S3 Object Management using AWS CLI

Objective:
Learn how to manage files (objects) inside an S3 bucket using AWS CLI.

---------------------------------------------------------------------

What is an S3 Object?

An object is a file stored inside an S3 bucket.

Examples:
- image.jpg
- notes.txt
- video.mp4
- backup.zip

Every object has:
- Data (the file)
- Key (file name/path)
- Metadata (information about the file)


---------------------------------------------------------------------

Bucket vs Object

Bucket:
- A container used to store objects.
- Similar to a folder.

Object:
- The actual file stored inside a bucket.

Example:

Bucket:
devops-day17-bucket

Objects:
- file1.txt
- file2.txt

---------------------------------------------------------------------

Commands Practiced

1. List Buckets

aws s3 ls

Shows all available S3 buckets.

-------------------------------------------------------

2. Create Bucket

aws s3 mb s3://devops-day17-bucket

Creates a new S3 bucket.

---------------------------------------------------------

3. Upload File

aws s3 cp file1.txt s3://devops-day17-bucket/

Uploads a file to the bucket.

------------------------------------------------------------

4. List Objects

aws s3 ls s3://devops-day17-bucket

Lists all files inside the bucket.

---------------------------------------------

5. Copy Object

aws s3 cp s3://devops-day17-bucket/file1.txt s3://devops-day17-bucket/file1-copy.txt

Creates a copy of an existing object.

------------------------------------------------------------

6. Download Object

aws s3 cp s3://devops-day17-bucket/file2.txt downloads/

Downloads an object from S3.

------------------------------------------------------

7. Delete Object

aws s3 rm s3://devops-day17-bucket/file1-copy.txt

Deletes an object from the bucket.

----------------------------------


Summary

Today I learned how to:
- Create an S3 bucket
- Upload files
- List bucket contents
- Copy objects
- Download objects
- Delete objects

These are the basic day-to-day operations performed on Amazon S3 using the AWS CLI.

AWS Day 5
---------

Topic:
AWS CLI

AWS CLI (Command Line Interface) allows us to manage AWS services from the terminal.

Benefits:
- Automate AWS tasks
- Script infrastructure
- Used in CI/CD pipelines
- Preferred by DevOps Engineers

Command:
aws --version

Result:
Verified AWS CLI installation.

----------------------------------------------------------------
AWS CLI Structure

aws <service> <command>

Examples:

aws ec2
aws s3
aws iam
aws configure

The CLI allows automation instead of using the web console.
